# Sales Analysis Dashboard (Tableau)

## Project Overview
This project analyzes sales data using Tableau dashboards to identify trends, product performance, and customer behavior.

## Tools Used
- Excel (Data Cleaning using Power Query)
- Tableau (Visualization & Dashboard)

## Dashboards
1. Executive Overview – Sales trends and country performance
2. Product Performance – Product line and deal size analysis
3. Sales Insights – Distribution, scatter, and heatmap analysis

## Key Insights
- Strong sales performance in Q4, especially in the USA
- Classic Cars generate the highest revenue
- Medium deal size contributes the most to total sales
- Majority of orders are low-value with few high-value outliers
- Sales peak in November indicating seasonal demand
